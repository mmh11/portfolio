# Portfolio Website

Interactive portfolio website built with React and Vite, featuring a 3D landing scene and a scrollable resume section. [sze.bio](https://sze.bio)

## Tech Stack

- React
- Vite
- MUI (Material UI)
- Spline (`@splinetool/react-spline`)

## 3D and Asset Credits

- **Spline** is used to render and embed the interactive 3D scene.
- **Tripo3D** was used in the 3D asset/model workflow.

- Images, icons, and 3D assets in this repository are used for personal portfolio presentation.

## Getting Started

### 1. Install dependencies

```bash
npm install
```

### 2. Run locally

```bash
npm run dev
```

### 3. Build for production

```bash
npm run build
```

### 4. Preview production build

```bash
npm run preview
```

## Deployment

This project is deployed on Vercel as [sze.bio](https://sze.bio)
